Make a Repeat, Alert, or Bookmark
1
You can create a repeating tag or an alert tag just as you would in any calendar.
2
If you create a repeat tag, you can set it to confirm each day. This means that you will need to click on the repeat tag for it to be logged as tracked.
3
You can also make a tag bookmark. What is the point of a tag bookmark? Simply, to reduce steps in tracking. Bookmarks are placed in the dock above your tracking tags. A bookmark can be pushed 0 to N times per day.





FORMAT:
[tag][amount*][unit*][timestamp*][(comment)*][repeat remind bookmark]
*optional
