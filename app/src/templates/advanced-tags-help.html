Format Advanced tags
Advanced tags
precise.ly supports a number of keywords for text-centric usage.

Keywords:
start (for durations)
midnight
stop (for durations)
repeat
begin (for durations)
remind (for creating an alert)
end
bookmark (for creating a bookmark)
noon
pinned (for creating a bookmark)
walk
button (for creating a bookmark)


Comments
Place tag comments at the end of a tag entry

Examples:
sleep 8 hours (really great)
indigestion 7am (egads, the buffet, a mistake)
cardio 30 min (cross-training)

Multi-dimensional

Examples:
resistance 4 intensity 32 minutes
meditation 2 effectiveness 30 minutes
box 3 inches width 2 inches height 5 inches depth

Durations

Examples:
sleep start 8pm
sleep stop 5am

Creating a duration currrently produces a tag that calculates the duration, much like a stopwatch. Tracking with sleep start and stop tags, in the examples above, would produce this tag: sleep 9 hours.
