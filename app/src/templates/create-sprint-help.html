Create Trackathon

First, what is a trackathon?

A trackathon is a set of tags that are tracked for a set length of time. You can invite others to your trackathon to compare results or go it alone. A trackathon is best driven by a research question.

Example Trackathon:

Does acupuncture affect sleep? Let's find out.
4 weeks without acupuncture to baseline. 2 months with acupuncture three days per week.
Length:
12 Weeks
Tags:
acupuncture, sleep, cardio, caffeine, alcohol
A trackathon can be inspired by a question or it can be some type of training regimen or anything you find interesting.

Example Trackathon:

Training for a 5K by incorporating cross-training 3 days per week.
Length:
8 Weeks
Tags:
regular run (2-4 miles), cross-train, rest day,long run (5-8 miles), weekly mileage,best time for the week

To create a trackathon, make sure you are in TRACKATHON
3
You may enter tags for the trackathon which appear as bookmarks, repeat daily, or create an alert daily to remind you to track the tag. For example, if you want to track headaches, just enter "headache" as a tag. If you want to be reminded to take aspirin every day at 8pm, enter "aspirin 200mg" and tap the "create daily alert" button.
4
You may invite participants to your trackathon with the "add participant" button.
