First, what is an experiment?

An experiment is a set of tags that are tracked for a set length of time. You can invite others to your experiment to compare results or go it alone. An experiment is best driven bya research question

Example Experiment:

Does acupuncture affect sleep? Let's find out.
4 weeks without acupuncture to baseline. 2 months with acupuncture three days per week.
Length:
12 Weeks
Tags:
acupuncture, sleep, cardio, caffeine, alcohol
An experiment can be inspired by a question or it can be some type of training regimen or anything you find interesting.

Example Experiment:

Training for a 5K by incorporating cross-training 3 days per week.
Length:
8 Weeks
Tags:
regular run (2-4 miles), cross-train, rest day,long run (5-8 miles), weekly mileage,best time for the week
