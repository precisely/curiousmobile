Create a tracking tag
1
Enter your tag in the text box at the top of the screen.
2
Example tags:
migraine 5 7:14am(is it my pillow?)
run 34 min 9am (really humid today)

FORMAT:
[tag][amount*][unit*][timestamp*][(comment)*][repeat remind button*]
*optional

**Make sure you're on the 'Track' tab**



Type your tag in the box provided above, and then set repeat and alerts, or create a button.



See Advanced Tracking for more information about how to make repeat, alert and button tags, s well as setting a repeat frequency, and an end date.
